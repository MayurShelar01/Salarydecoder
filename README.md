# Salary Decoded

Mobile-first salary calculator for Indian freshers.

## Tech stack

- Vite
- React 18
- TypeScript (strict)
- Tailwind CSS
- shadcn/ui (primitives under `src/components/ui/`)
- Zustand
- react-hook-form + Zod
- Vitest + Testing Library

## Getting started

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Run tests (Module 1 tests are expected to fail):

```bash
npm run test
```

## Folder structure (Module 1)

- `src/components/ui/`: shadcn primitives only
- `src/components/layout/`: layout placeholders
- `src/lib/config/`: constants + FY 2025-26 slab config
- `src/lib/utils/`: shared helpers (`cn`, INR formatters)
- `src/lib/types/`: TypeScript types
- `src/store/`: Zustand store
- `src/tests/`: red-stage specs for Module 2 (intentionally failing)

# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
